## 实验阶段

### 1. 指导书基础复现

对应材料：

- `misc/images/`
- `training_scripts/train.py`
- `evaluation_scripts/test.py`
- `models_old/model.pkl`
- 当前保留模型：`../models/model_0-5test_own.pkl`

说明要点：

- 按指导书原始 CNN 训练 0-5 手势数字。
- 完成 Web 端基本识别，并保留 `train.py/test.py/inference.py` 主流程。
- 指导书数据分辨率较低，复杂摄像头场景泛化不足。

### 2. 数据集扩展

对应材料：

- `misc/extra_data/Chinese-number-gestures-recognition/`
- `dataset_builders/build_chinese_manifest.py`
- `dataset_builders/build_chinese_dataset_txt.py`
- `misc/extra_data/chinese_gesture_manifest_0_9.csv`

说明要点：

- 中国数字手势数据文件名形如 `6_1_0_167.jpg`，第一个数字为真实标签。
- 将数据构建为指导书风格的 `train.txt/test.txt`。
- 发现随机切分存在同源样本泄漏，因此改为按来源组切分。

### 3. 多模型训练对比

对应脚本：

- `training_scripts/train_chinese_cnn_aug_gpu.py`
- `training_scripts/train_chinese_efficientnet_grouped_gpu.py`
- `training_scripts/train_chinese_resnet34_grouped_gpu.py`
- `training_scripts/train_chinese_mobilenet_grouped_gpu.py`

关键结果：

- 增强 CNN 0-9：grouped test 约 56.07%。
- EfficientNet-B0 0-9：grouped test 约 76.29%。
- ResNet34 0-9：grouped test 约 82.79%。

指标与日志：

- `misc/outputs/chinese_cnn_aug_stdout.txt`
- `misc/outputs/chinese_efficientnet_b0_grouped_metrics.json`
- `misc/outputs/chinese_resnet34_grouped_metrics.json`

### 4. ROI 与复杂场景优化

对应代码：

- 当前根目录：`../hand_detector.py`
- 历史评估：`evaluation_scripts/evaluate_models.py`
- 复杂样本测试：`evaluation_scripts/synthetic_clutter_test.py`

说明要点：

- 原始摄像头画面背景复杂，模型容易受到桌面、衣服、人脸等干扰。
- 实现多 ROI 候选、肤色/边缘候选、背景弱化和模型置信度重排序。
- 将“硬分割”改为更保守的 ROI 裁剪和背景弱化，减少分割失败带来的误识别。

### 5. 融合模型优化

最终模型：

- `../models/GestureDigitRecognizer_model.pt`
- `../models/GestureDigitRecognizer_model.json`

说明要点：

- 不再只依赖单个模型，而是结合多个模型的逐类优势。
- 根据实测表现提高特定模型在特定数字上的权重。
- 对 3、7、8/2 等易混数字增加后处理规则。
- 将最终融合规则和基础模型打包为单文件模型，命名为“手势数字识别模型”。

### 6. Web 展示优化

对应文件：

- `../web/index.html`
- `../app_server.py`

说明要点：

- Web 端增加上传图片和摄像头实时识别。
- UI 改为全屏工具布局，支持浅色/深色主题。
- 默认隐藏底层模型细节，只展示最终识别结果。
- 调试模式 `?debug=1` 可展示多模型对比，用于报告说明融合过程。
- 早期 App/PWA 尝试已归档到 `misc/app_archived/`，当前 GitHub 项目以 Web 端作为扩展成果。

## 报告可引用结论

1. 单一 CNN 在复杂场景泛化不足，数据集分辨率和背景差异是主要问题。
2. 新数据集必须避免同源增强样本同时出现在训练集和测试集。
3. 高分辨率迁移模型（ResNet34、EfficientNet-B0）在 grouped test 上明显优于小 CNN。
4. 实际摄像头落地时，仅看整体准确率不够，需要结合 ROI、逐类表现和实时反馈调参。
5. 最终采用多模型融合，比单模型更适合处理不同数字手势的差异。
